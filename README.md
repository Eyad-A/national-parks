# National Parks
## An app that allows you to search for US national parks and view information such as location and ongoing events. 

### API used: 
[National Park Service](https://www.nps.gov/subjects/developer/index.htm) 

### Tech used: 
- Python
- Flask
- SQLAlchemy
- Bootstrap
